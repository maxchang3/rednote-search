让小红书更像一个专注搜索的工具，目前功能包括：

- 隐藏首页时间线和干扰内容，专注搜索
- 移除侧边导航中的直播、发布、通知入口
- 隐藏搜索框下方的「猜你想搜」提示
- 隐藏通知按钮的未读提示
- 支持按 `/` 快速聚焦搜索框
- ...

所有功能均可在设置中开启或关闭。
